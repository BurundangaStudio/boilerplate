# Vue 2 simple website (V1.0)

Vue 2 most simple website with routes, transitions, http requests and vuex.

### Technologies

* Vue 2.1
* Vuex 2.0
* VueRouter 2.1.1
* VueResource 1.0.3
* SCSS
* GSAP - TweenMax
* Webpack
* ES2015 | ES6

### Usage

`npm install` +
`npm run dev`

### Production

`npm run build`
